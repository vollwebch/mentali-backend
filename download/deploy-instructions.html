🧠 Mentali App - Instrucciones de Despliegue

La app está lista para desplegar. El código fuente está en GitHub:

https://github.com/vollwebch/mentali-app

Opción 1: Netlify (Recomendado)

Paso 1: Ve a netlify.com y crea una cuenta gratis

Paso 2: Haz clic en "Add new site" → "Import an existing project"

Paso 3: Conecta tu GitHub y selecciona el repositorio vollwebch/mentali-app

Paso 4: Configura el build:

Build command: npm install && npx expo export --platform web
Publish directory: dist

Paso 5: Haz clic en "Deploy site" y espera 2-3 minutos

Opción 2: Vercel

Paso 1: Ve a vercel.com y crea una cuenta

Paso 2: Haz clic en "Add New" → "Project"

Paso 3: Importa el repositorio vollwebch/mentali-app

Paso 4: Vercel detectará automáticamente la configuración del vercel.json

Paso 5: Desactiva "Deployment Protection" en Settings → Deployment Protection

Opción 3: GitHub Pages

Paso 1: Ve a Settings del repositorio en GitHub

Paso 2: En "Pages", selecciona "GitHub Actions" como source

Paso 3: Crea un workflow que ejecute:

npm install
npx expo export --platform web
