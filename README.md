# 🌱 Mentali - App de Red Social para Pensamientos

Una aplicación móvil para compartir pensamientos, reflexiones y emociones de forma anónima y temporal.

## ✨ Características

### 🧠 **Gestión de Pensamientos**
- Escribir y compartir pensamientos con emojis emocionales
- Pensamientos temporales que expiran automáticamente
- Modo privado para reflexiones personales
- Respuestas de IA para insights emocionales

### 🏆 **Sistema de Premios y Gamificación**
- 100+ premios desbloqueables por actividad
- Árbol emocional que crece con tu progreso
- Rachas de días y logros especiales
- Popups con sonido y vibración al desbloquear

### 🤖 **Psicólogo Virtual con IA**
- Chat inteligente con respuestas empáticas
- Apoyo emocional sin juicios
- Detección de emociones y temas
- Consejos personalizados

### 🎨 **Interfaz Moderna**
- Diseño oscuro/claro automático
- Animaciones fluidas y feedback háptico
- Filtros por emociones
- Navegación intuitiva

### 📊 **Análisis Personal**
- Historial privado de pensamientos
- Análisis de patrones emocionales
- Estadísticas de progreso
- Exportación de diario

## 🚀 Instalación

### Prerrequisitos
- Node.js 18+
- npm o yarn
- Expo CLI
- Expo Go (para probar en móvil)

### Pasos de Instalación

1. **Clonar el repositorio**
```bash
git clone https://github.com/vollwebch/mentali-app.git
cd mentali-app
```

2. **Instalar dependencias**
```bash
npm install
```

3. **Iniciar la aplicación**
```bash
npm start
```

4. **Probar en dispositivo**
- Escanea el código QR con Expo Go (Android)
- Escanea con la app Cámara (iOS)
- O presiona 'w' para abrir en web

## 🛠️ Scripts Disponibles

```bash
npm start          # Iniciar servidor de desarrollo Expo
npm run android    # Abrir en Android
npm run ios        # Abrir en iOS
npm run web        # Abrir en navegador
npm run dev        # Desarrollo con Next.js (API routes)
npm run build      # Build para producción
```

## 📱 Uso de la Aplicación

### **Escribir Pensamientos**
1. Toca el botón flotante (+)
2. Selecciona una emoción
3. Escribe tu pensamiento
4. Elige duración y visibilidad
5. Publica

### **Chat con Psicólogo IA**
1. Ve a la pestaña "Psicólogo"
2. Escribe cómo te sientes
3. Recibe apoyo y reflexiones
4. Conversación privada y empática

### **Sistema de Premios**
- Los premios se desbloquean automáticamente
- Ve a "Premios" para ver tu colección
- El árbol emocional crece con tu actividad
- Configura sonido y vibración en ajustes

### **Navegación**
- **Inicio**: Ver pensamientos de la comunidad
- **Premios**: Tu colección de logros
- **Psicólogo**: Chat con IA
- **Perfil**: Tu espacio personal

## 🎯 Tecnologías Utilizadas

- **React Native** + **Expo**
- **TypeScript**
- **Expo Router** (navegación)
- **Next.js** (API routes para IA)
- **Expo AV** (audio)
- **Expo Haptics** (vibración)
- **AsyncStorage** (persistencia)
- **Linear Gradient** (efectos visuales)
